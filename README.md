### bio_anaylis
-----
##### TCGA下载数据处理及差异分析
----
* DESeq2_used.R、edgeR_used.R和limma_used.R为三个R脚本，分别是利用DESeq2、edgeR和limma三种包做的差异分析代码；
* genenote.R是做基因注释、数据处理的R脚本；file_build.py是数据批量整合的python脚本;
